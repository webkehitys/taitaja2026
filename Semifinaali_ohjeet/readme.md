## Sallitut teknologiat ja työkalut

**Frontend:** HTML, CSS, JavaScript tai React / Vue / Angular / Next
**Backend:** PHP (natiivi tai Laravel) tai Node.js (Express.js)

Valmiiden UI-komponenttikirjastojen käyttö **ei ole sallittua.**

Tekoälyn käyttö on sallittua ja siitä on ilmoitettava erillisen ohjeen mukaisesti.
