### Kaavaehdotusprosessissa on kuusi vaihetta

1. Aloitus
   → OAS, aloituspäätös
2. Valmistelu
   → Kaavaluonnos, selvitykset
3. **Kuuleminen**
   → Mielipiteet, kuulutus
4. Ehdotus
   → Kaavaehdotus, selostus
5. **Nähtävilläolo**
   → 30 vrk, Muistutukset, lausunnot, vastineet
6. Hyväksyminen
   → Hyväksymispäätös, kuulutus, lainvoima

Kaavaehdotuksia voi kommentoida vaiheissa **Kuuleminen** sekä **Nähtävilläolo**

Jos et vielä osaa käyttää tietokantaa, voit käyttää tätä tietoa hyödyksi.

### Planförslagsprocessen består av sex steg

1. **Initiering** (“Aloitus”)
   → Deltagande- och bedömningsplan (OAS), beslut om inledning

2. **Förberedelse** (“Valmistelu”)
   → Planutkast, utredningar

3. **Samråd** (“Kuuleminen”)
   → Åsikter, kungörelse

4. **Förslag** (“Ehdotus”)
   → Planförslag, planbeskrivning

5. **Offentligt framläggande** (“Nähtävilläolo”)
   → 30 dagar, anmärkningar, utlåtanden, bemötanden

6. **Godkännande** (“Hyväksyminen”)
   → Godkännandebeslut, kungörelse, laga kraft

Planförslag kan kommenteras under stegen **Samråd / Kuuleminen** och **Offentligt framläggande / Nähtävilläolo**.

Om du ännu inte kan använda en databas kan du dra nytta av denna information.

### The planning proposal process consists of six stages

1. Initiation (“Aloitus”)
   → Participation and Assessment Plan (OAS), decision to initiate
2. Preparation (“Valmistelu”)
   → Draft plan, studies
3. **Consultation** (“Kuuleminen”)
   → Opinions, public notice
4. Proposal (“Ehdotus”)
   → Planning proposal, explanatory report
5. **Public display** (“Nähtävilläolo”)
   → 30 days, Objections, statements, responses
6. Approval (“Hyväksyminen”)
   → Approval decision, public notice, legal validity

Planning proposals can be commented on during the **Consultation / Kuuleminen** and **Public display / Nähtävilläolo** stages.

If you are not yet familiar with using a database, you can make use of this information.
